# Simulatore Esame Psichiatria 🧠

Questo progetto è un simulatore di quiz basato su una banca dati di domande di psichiatria, distribuito tramite Streamlit.

## Struttura del Progetto
- `quiz_app.py`: L'applicazione principale Streamlit.
- `banca_domande.json`: Database delle domande in formato JSON.
- `requirements.txt`: Dipendenze Python necessarie.

## Manutenzione e Policy Cleanup
Per mantenere la repository pulita e ordinata, è stabilita la seguente regola per gli script di manutenzione:

> **Policy Script Temporanei**: Tutti gli script di utilità o di correzione "one-shot" (come `fix_banca.py`, `temp_fix_typos.py`, o simili) devono essere **eliminati immediatamente dopo il loro utilizzo**.

Questi script non devono essere caricati in modo permanente nella repository se il loro compito è terminato.
